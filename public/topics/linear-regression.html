What is Regression Analysis?

In machine learning, regression analysis is a statistical technique that predicts continuous numeric values based on the relationship between independent and dependent variables. The main goal of regression analysis is to plot a line or curve that best fits the data and to estimate how one variable affects another.

Regression analysis is a fundamental concept in machine learning and it is used in many applications such as forecasting, predictive analytics, and more.

In machine learning, regression is a type of supervised learning. The key objective of regression-based tasks is to predict output labels or responses — which are continuous numeric values — for the given input data. The output will be based on what the model has learned in the training phase.

Regression models use the input data features (independent variables) and their corresponding continuous numeric output values (dependent or outcome variables) to learn specific associations between inputs and corresponding outputs.

Terminologies Used in Regression Analysis

Let us understand some basic terminologies used in regression analysis before going into further detail.

Independent Variables

Used to predict the value of the dependent variable. Also called predictors. In a dataset, these are represented as features.

Dependent Variables

Variables whose values we want to predict. These are the main factors in regression analysis — represented as target variables.

Regression Line

A straight line or curve that a regressor plots to best fit the data points in the feature space.

Overfitting

When the model performs well on training data but poorly on test data. Also referred to as the problem of high variance.

Underfitting

When the model doesn't perform well even on training data. Also referred to as the problem of high bias.

Outliers

Data points that don't fit the pattern of the rest of the data — extremely high or extremely low values in the dataset.

Multicollinearity

Occurs when independent variables (features) have dependencies among themselves, which can distort model coefficients.

How Does Regression Work?

Regression in machine learning is a supervised learning approach. Basically, regression is a statistical technique that finds a relationship between dependent and independent variables. To implement regression in machine learning, a regression algorithm is trained with a labeled dataset containing features (independent variables) and target values (dependent variables).

During the training phase, the regression algorithm learns the relationship between independent variables (predictors) and dependent variables (target). The regression models then predict new values based on the learned relationship between predictors and targets.

Types of Regression in Machine Learning

Generally, the classification of regression methods is done based on three metrics — the number of independent variables, type of dependent variables, and shape of the regression line. Below are the commonly used types of regression:

01

Linear Regression

Analyzes a linear relationship between a dependent variable and independent variables. Divided into simple and multiple linear regression.

02

Logistic Regression

Predicts the probability of an event occurring. Uses a sigmoid function to map outputs between 0 and 1. Ideal for binary classification.

03

Polynomial Regression

Models the relationship as an n-th degree polynomial. Captures non-linear relationships beyond simple linear regression.

04

Lasso Regression

L1 regularization technique that adds the sum of absolute coefficients as a penalty. Prevents overfitting on high-dimensional data.

05

Ridge Regression

L2 regularization that adds the square of coefficient magnitudes as a penalty. Reduces complexity and handles multicollinearity.

06

Decision Tree Regression

Splits data into subsets based on input feature values, assigning each subset a numeric prediction value.

07

Random Forest Regression

Ensemble of decision trees using bagging. Combines multiple trees to improve accuracy and reduce variance.

08

Support Vector Regression

Uses SVM to solve regression problems. Handles linear and non-linear relationships; highly resistant to outliers.

1. Linear Regression

Linear regression is the most commonly used regression model in machine learning. It analyzes the linear relationship between a dependent variable with a given set of independent variables. A linear relationship means when the value of one or more independent variables changes, the dependent variable changes accordingly.

Linear regression is further divided into two subcategories: simple linear regression and multiple linear regression (multivariate).

In simple linear regression, a single independent variable is used to predict the dependent variable.

Y = mX + b
  • Y — the dependent variable we are trying to predict
  • X — the independent variable used to make predictions
  • m — the slope of the regression line (effect of X on Y)
  • b — the Y-intercept (value of Y when X = 0)

Simple Linear Regression — Scatter Plot with Regression Line

2. Logistic Regression

Logistic regression is a popular machine learning algorithm used for predicting the probability of an event occurring. It is a generalized linear model where the target variable follows a Bernoulli distribution. Logistic regression uses a logistic (sigmoid) function to learn the relationship between independent and dependent variables. The sigmoid function produces a probability between 0 and 1.

It is mostly used in binary classification problems, where the target variable is categorical with two classes.

Sigmoid Function — Logistic Regression Output Curve

3. Polynomial Regression

Polynomial regression models the relationship between the independent variable and the dependent variable as an n-th degree polynomial function. It allows for more complex relationships to be captured, beyond the linear relationship in simple regression. It is widely used for non-linear regressions and is more sensitive to outliers.

Polynomial Regression — Degree 1 vs Degree 3

4. Lasso Regression

Lasso regression is a regularization technique that uses a penalty to prevent overfitting and improve accuracy. It performs L1 regularization, modifying the loss function by adding the penalty equivalent to the summation of the absolute value of coefficients. It is often used to handle high-dimensional and high-correlation data.

5. Ridge Regression

Ridge regression is a regularization technique that performs L2 regularization. It modifies the loss function by adding the penalty equivalent to the square of the magnitude of coefficients. It helps reduce model complexity, improve prediction accuracy, and corrects multicollinearity in regression analysis.

L1 (Lasso) vs L2 (Ridge) — Coefficient Shrinkage

6. Decision Tree Regression

Decision tree regression uses the decision tree algorithm to predict numerical values. It works by splitting the data into smaller subsets based on input feature values and assigning each subset a numerical value. The tree fits local linear regressions that approximate a curve, and each leaf represents a numeric value. The algorithm tries to reduce the Mean Square Error at each child node.

7. Random Forest Regression

Random forest regression is a supervised machine learning algorithm that uses an ensemble of decision trees to predict continuous target variables. It uses a bagging technique involving randomly selecting subsets of training data to build smaller decision trees. These smaller models are combined into a random forest model that outputs a single prediction value, improving accuracy and reducing variance.

Decision Tree vs Random Forest — Prediction Comparison

8. Support Vector Regression

Support Vector Regression (SVR) uses support vector machines to solve regression problems. It can learn non-linear relationships between input data (feature variables) and output data (target values). SVR handles linear as well as non-linear relationships in datasets, is resistant to outliers, and has high prediction accuracy.

Types of Regression Models

Simple Regression Model

The most basic regression model in which predictions are formed from a single, univariate feature of the data.

Multiple Regression Model

Predictions are formed from multiple features of the data, allowing for more complex real-world modeling.

Evaluation Metrics for Regression

Common evaluation/performance metrics for regression models:

Metric Formula / Description Interpretation
MAE — Mean Absolute Error Average of |predicted − actual| Lower is better. Robust to outliers.
MSE — Mean Squared Error Average of (predicted − actual)² Lower is better. Penalises large errors heavily.
Median Absolute Error Median of |predicted − actual| Very robust to outliers.
RMSE — Root MSE √MSE Same units as target variable.
R² Score 1 − (SS_res / SS_tot) Best = 1.0; can be negative for very poor models.
MAPE 100 × mean(|actual − pred| / |actual|) Percentage equivalent of MAE.

How to Select the Best Regression Model?

Key Consideration: Evaluate model performance using metrics such as MSE, MAE, and R-squared. Compare models like linear regression, decision trees, and random forests, then choose the model with the highest performance metrics, the lowest complexity, and the best interpretability for your specific use case.

You can consider factors like performance metrics, model complexity, interpretability, and the size/type of your dataset to select the best regression model. No single model is universally the best.

Full Combined Python Script

linear_regressor_full.py
$ python linear_regressor_full.py

# ── Imports ──────────────────────────────
import numpy as np
from sklearn import linear_model
import sklearn.metrics as sm
import matplotlib.pyplot as plt

# ── Load Data ────────────────────────────
input_data = np.loadtxt(r'C:\linear.txt', delimiter=',')
X, y = input_data[:, :-1], input_data[:, -1]

# ── Train/Test Split ─────────────────────
n = int(0.6 * len(X))
X_train, y_train = X[:n], y[:n]
X_test, y_test = X[n:], y[n:]

# ── Train Model ──────────────────────────
reg = linear_model.LinearRegression()
reg.fit(X_train, y_train)
y_pred = reg.predict(X_test)

# ── Visualise ────────────────────────────
plt.scatter(X_test, y_test, color='red')
plt.plot(X_test, y_pred, color='black', linewidth=2)
plt.show()

# ── Metrics ──────────────────────────────
print(f"MAE : {sm.mean_absolute_error(y_test, y_pred):.2f}")
print(f"MSE : {sm.mean_squared_error(y_test, y_pred):.2f}")
print(f"RMSE : {np.sqrt(sm.mean_squared_error(y_test, y_pred)):.2f}")
print(f"R² : {sm.r2_score(y_test, y_pred):.2f}")

Applications of Linear Regression

Linear Regression is one of the most practical algorithms in Machine Learning — it appears everywhere from everyday predictions to advanced AI systems.

1. Real-World Applications (Basic to Industry)

💰 Finance

  • Predict stock price trends
  • Estimate house prices
  • Risk modeling
Example: Predict house price based on area, location, and rooms.

🌾 Agriculture (useful for your project)

  • Crop yield prediction
  • Fertilizer recommendation
  • Weather-based production
Example: \( \text{Yield} = w_1\cdot(\text{rainfall}) + w_2\cdot(\text{soil quality}) + w_3\cdot(\text{temperature}) \)

🏢 Business Analytics

  • Sales forecasting
  • Customer demand prediction
  • Marketing ROI estimation

🚗 Engineering

  • Predict fuel consumption
  • Performance optimization

2. Linear Regression in Deep Learning

Linear Regression is the foundation of neural networks.

Inside a neuron: Output = \(w_1x_1 + w_2x_2 + \dots + b\)
Key insight: Linear regression = a single neuron. Deep learning = many stacked regressions + activation functions.
  • Used in the final layer for regression tasks
  • MSE loss is a standard training objective for regression
  • Weight updates rely on gradient descent (built on the same linear structure)

3. Linear Regression in NLP

Even in NLP, regression can be used when the output is a score or continuous value.

  • Sentiment score prediction: predict a score between -1 and 1
  • Text complexity: predict readability level
  • Embedding projection: map text embeddings to numeric outputs
Example: Sentiment Score = \(w_1\cdot(\text{word}_1) + w_2\cdot(\text{word}_2) + \dots\)

4. Linear Regression in Generative AI (Gen AI)

Generative AI is mostly deep learning, but regression-style layers are still used for scoring and calibration tasks.

  • Output calibration: fine-tune scores/probabilities
  • Reward models (RLHF): predict a “goodness score” for an output
  • Embedding mapping: transform vectors between spaces
Study tip: If you understand Linear Regression deeply (loss, gradients, and weights), you’ll find it much easier to learn Neural Networks, Deep Learning, and even RLHF concepts later.
