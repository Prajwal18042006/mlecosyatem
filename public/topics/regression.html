REGRESSION

Regression analysis is a statistical technique used to predict continuous numeric values based on relationships between variables.

What is Regression?

Regression is a supervised learning technique where the model learns relationships between independent (input) and dependent (output) variables.

Linear Regression Visualization

Key Terminologies

  • Independent Variables: Inputs / features
  • Dependent Variable: Target output
  • Regression Line: Best fit line
  • Overfitting: Model fits training too well
  • Underfitting: Model too simple
  • Outliers: Extreme values
  • Multicollinearity: Feature dependency

How Regression Works

Regression learns from labeled data and predicts continuous values based on learned patterns.

Types of Regression

  • Linear Regression
  • Logistic Regression
  • Polynomial Regression
  • Lasso Regression
  • Ridge Regression
  • Decision Tree Regression
  • Random Forest Regression
  • Support Vector Regression

Polynomial Regression Curve

Linear Regression Formula

Y = mX + b

  • Y = Output
  • X = Input
  • m = Slope
  • b = Intercept

Evaluation Metrics

  • MAE (Mean Absolute Error)
  • MSE (Mean Squared Error)
  • RMSE
  • R² Score

Error Metrics Comparison

Applications

  • Forecasting (GDP, prices)
  • Finance risk prediction
  • Business optimization
  • Economics

Python Implementation

import numpy as np
from sklearn import linear_model
import matplotlib.pyplot as plt

X = np.array([[1],[2],[3],[4]])
y = np.array([2,4,6,8])

model = linear_model.LinearRegression()
model.fit(X,y)

pred = model.predict([[5]])
print(pred)

Model Evaluation Output

Mean Absolute Error = 1.78
Mean Squared Error = 3.89
R2 Score = -0.09
